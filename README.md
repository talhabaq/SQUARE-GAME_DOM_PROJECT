This is a mini project of DOCUMENT OBJECT MODEL (DOM) using javascript , html and css.
It has following attributes.
1. when mouse enters block 1 it generates random value from 1 to 100 AND when mouse leaves it again changes to 1.
2. when mouse enters block 2 it changes into one of the following colour (red,blue,green) AND when mouse leaves the box the color again becomes white.
3. when mouse enters block 3 it gains a random colour AND when mouse leaves box again turns white
4. when mouse enters block 4 it turns all other boxes into random colorful boxes but with different colors AND remains white when mouse leaves
